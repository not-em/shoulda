# shoulda

Pick a year, a stock, and an amount and see what you could've earned.

---

## Customising the ticker list

Edit `ticker.py` to change the list of tickers. 
---

## File structure

```
├── index.html          ← the app
├── data.json           ← price data (generated)
├── meta.json           ← search index (generated)
├── download_data.py    ← run once to generate the above
├── ticker.py           ← list of tickers (SP500 + major indices)
└── README.md
```
